| Tip | Example |
|-----|---------|
| Be specific about genre | "Electronic dance music with heavy bass drops" |
| Describe the mood | "Melancholic piano piece, slow tempo, emotional" |
| Mention instruments | "Acoustic guitar with light percussion" |
| Set the tempo | "Upbeat, fast-paced, 140 BPM energy" |
| Add context | "Music for a coffee shop, background ambiance" |

|----------|-------------|

### Best Practices

| Practice | Description |
|----------|-------------|
| Be descriptive | Include style, colors, mood, composition |
| Use art terms | "Digital art", "album cover style", "cinematic" |
| Specify format | "Square aspect ratio", "high detail" |
| Avoid text | AI struggles with text in images |

## Workshop Materials

### University of Chicago Vibe Coding Workshop

| Resource | Description |
|----------|-------------|
| **Instructor** | [Agnel Nieves](https://agnelnieves.com) |
| **Project** | AI Playlist Generator |
| **Focus** | Modern AI-powered web development |

### What You'll Learn

1. **Next.js 16** - Latest React framework features
2. **AI Integration** - Working with multiple AI APIs
